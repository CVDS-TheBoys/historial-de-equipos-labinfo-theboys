Asociar elementos





Elementos disponibles











Selecciona un equipo y asocia elementos a él. Si no seleccionas un elemento para asociar, el equipo no tendrá cambio alguno en dicho elemento. Si seleccionas un elemento y el equipo ya tenía uno de ese tipo, se desasociará el que estaba y quedará disponible para asociarlo a otro equipo.
